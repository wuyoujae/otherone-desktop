Quick Start Guide

Learn how to set up Otherone and run your first autonomous agent workflow in less than 5 minutes.

Welcome to Otherone. Unlike traditional AI chatbots, Otherone acts as a digital worker that operates directly on your operating system. It can read your files, execute terminal commands, and browse the web autonomously to accomplish complex goals.

Prerequisite: Before proceeding, ensure you have downloaded and installed the latest version of the Otherone desktop application for your specific OS (macOS, Windows, or Linux).

1. Initial Setup

Once you launch the application, you'll be greeted by the API configuration screen. Otherone relies on Large Language Models (LLMs) to power its reasoning engine.

  • Navigate to Settings > Models & Engines.
  • Add your provider's API Key (e.g., OpenAI, Anthropic, or DeepSeek).
  • Alternatively, if you prefer complete privacy, select the Ollama Local Model option.

2. Running Your First Task

Let's ask the agent to create a simple React component and save it to your desktop. Open the main chat interface and paste the following prompt:

Prompt 
# Tell the agent to interact with your file system
Please create a new folder named "OtheroneDemo" on my Desktop.
Inside it, write a standard React Login Component named Login.jsx
Make sure to include basic CSS styling.

Press Enter to submit. You will instantly see the Agent's thought process unfold in the UI as it plans the steps, navigates your directory, and writes the file.

3. Monitoring Workflows

For tasks that take longer to complete (like analyzing large codebases), Otherone will move the process to the background. You can monitor all active executions in the Workflows tab on the left sidebar.
